A zero-dependency, single-file local web application for visualizing and exploring session logs generated by the [Claude Code CLI](https://docs.anthropic.com/en/docs/agents-and-tools/claude-code/overview).

When you use Claude Code, it stores a rich history of your interactions, including tool executions, internal reasoning, and terminal commands, in `.jsonl` (JSON Lines) files. Staring at raw stringified JSON is painful. This tool converts those logs into a beautiful, readable, chat-like interface.

## âś¨ Features

- **Zero Dependencies:** It's just one `index.html` file. No `npm install`, no build steps, no external frameworks.
- **100% Private & Local:** All parsing happens in your browser's memory using the local `FileReader` API. Your logs are **never** uploaded to a server or sent over the internet.
- **Cleans up the Noise:** Automatically filters out internal CLI queue operations and sidechains, showing you the clean, linear path of the conversation.
- **Smart Formatting:** Beautifully formats Markdown, code blocks, and Claude's tool inputs/outputs (like `Bash` commands, file reads, and file edits).
- **Claude's Internal Monologue:** View Claude's hidden `thinking` blocks to see exactly how it planned its actions.
- **Light & Dark Mode:** Built-in theme switcher to save your eyes during late-night coding sessions.
- **Secure:** Includes strict HTML escaping to prevent Cross-Site Scripting (XSS) if you are viewing shared logs.

## đźš€ How to Use

1. Clone this repository or simply download the `index.html` file.
2. Double-click `index.html` to open it in your web browser (Chrome, Firefox, Brave, Safari, etc.).
3. Click the upload box and select a `.jsonl` log file from your Claude Code sessions.
   * *Note: Claude Code typically stores session logs in `.claude/sessions/` within your project directory, or you can export them directly.*
4. Explore the conversation tree!

- Your source code and file contents
- Environment variables or API keys (if they were printed to the terminal)
- Private conversation history
